# Personal Site

A modern portfolio website built with Next.js.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
personal-site/
├── app/
│   ├── layout.js
│   ├── page.js
│   └── globals.css
├── public/
├── package.json
└── README.md
```

## Features

- Built with Next.js 14
- Responsive design
- Fast and optimized

## Customization

Edit `app/page.js` to start building your portfolio!

## Deployment

Deploy easily on [Vercel](https://vercel.com) or [GitHub Pages](https://pages.github.com/).
